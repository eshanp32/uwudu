## Installation
### Using Homebrew

If you use Homebrew, you can install `uwudu` with the following commands:

```bash
# Tap the uwudu formulae
brew tap uwudu/uwudu
brew install uwudu
```

### Using Pip

If you use Homebrew, you can install `uwudu` with the following commands:

```shell
pip install uwudu
export PATH=$PATH:$HOME/go/bin  #to add path, restart terminal or do `source ~/.xyzrc`
```
